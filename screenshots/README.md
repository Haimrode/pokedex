# Screenshots

Mets ici les captures d'écran de l'app, référencées depuis le README racine :

| Fichier attendu | Écran |
|---|---|
| `list.png` | Liste du Pokédex (avec quelques chips de type visibles) |
| `detail.png` | Fiche détail d'un Pokémon (sprite, stats, FAB favori) |
| `favorites.png` | Onglet Favoris avec au moins 2-3 Pokémons |

## Comment prendre les captures

### Sur ton téléphone Android
- **Volume bas + Power** simultanément
- L'image est dans `Internal Storage > DCIM > Screenshots` (ou `Pictures > Screenshots` selon le constructeur)
- Récupère-les via USB ou Google Photos / Drive
- Renomme et place-les ici

### Depuis Android Studio (si tu testes sur émulateur)
- Avec l'app lancée, dans la barre de l'émulateur clique sur l'icône **caméra** (📷)
- L'image apparaît dans `~/Desktop` ou un autre dossier selon ta config
- Ou utilise **Logcat → Screen Capture** pour exporter depuis l'IDE

## Format recommandé

- `.png` (lossless, meilleur rendu)
- Taille portrait : ~1080×2400 (résolution native de la plupart des tels) → ~1-3 MB par image
- Si trop lourd, redimensionne à 720px de large via TinyPNG ou Photoshop / Paint.NET
